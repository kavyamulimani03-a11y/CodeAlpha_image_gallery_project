# Image Gallery Project

## 📌 Project Overview

The Image Gallery Project is a responsive web application that displays a collection of images in a clean gallery layout. Users can click on images to view them in a lightbox with next and previous navigation. The project is built using HTML, CSS, and JavaScript.

## 🚀 Features

* Responsive image gallery layout
* Hover effects with smooth transitions
* Lightbox view for image preview
* Next and Previous navigation buttons
* Mobile-friendly responsive design
* Clean and modern UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## ⚙️ How to Run the Project

1. Download or clone this repository.
2. Open the project folder in a code editor.
3. Open the `index.html` file in a web browser.

### Using Live Server (Recommended)

1. Open the project in VS Code.
2. Install the Live Server extension.
3. Right-click `index.html`.
4. Click **Open with Live Server**.

## 📸 Functionality

* The gallery displays multiple images in a grid layout.
* Clicking on an image opens it in a lightbox view.
* Users can navigate between images using next and previous buttons.
* Hover effects improve user interaction and experience.
